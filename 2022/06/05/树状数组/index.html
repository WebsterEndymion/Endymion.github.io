树状数组原理

2022/6/5 算法 树状数组

树状数组

普通数组单点修改复杂度为O(1)，区间求和复杂度为O(n)。使用前缀和单点修改复杂度为O(n)，区间求和为O(1)。树状数组采用折中的方法。

对 [a,b] 进行区间查询只需查询 [1,b] 和 [1,a] 然后相减，可以把区间查询问题转化为求前n项和的问题。
树状数组就是这样一种结构，它巧妙地利用了二进制）。例如11，转化为二进制数就是 [1011] ，要求前11项和，可以分别查询 [0000,1000] 、[1000,1010]以及[1010,1011]的和，再相加

优点：修改查询复杂度都是O(logN)，比线段树系数少，比传统数组快且易于修改。
缺点：不能解决复杂区间问题。

代码

[Ci] 维护区间 (A*(i-lowbit(i))*,Ai]
lowbit(x) = x & -x;

单点修改

int tree[MAXN];
inline void update(int i, int x)
{
    for (int pos = i; pos < MAXN; pos += lowbit(pos))
        tree[pos] += x;
}

求前n项和

inline int query(int n)
{
    int ans = 0;
    for (int pos = n; pos; pos -= lowbit(pos))
        ans += tree[pos];
    return ans;
}

区间查询

inline int query(int a, int b)
{
    return query(b) - query(a - 1);
}
